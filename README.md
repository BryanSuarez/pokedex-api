# Levantar proyecto
1. Clonar el repo
2. Ejecutar
```
  yarn install
```
3. Tener Nest JS CLI instalado

4. Ejecutar
```
  yarn start:dev
```

5. Levantar base de datos
```
  docker-compose up -d
```

##Stack usado
* NestJS
* MongoDB
* TypeORM
* PostgreSQL
* Docker
* Docker Compose
